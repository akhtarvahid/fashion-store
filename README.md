# Fashion-store

## Description

App repo is private because .env.prod, .env.dev is pushed in repo.

Fashion-store is [LIVE](https://fashion-store.up.railway.app)

## Project setup

```bash
$ npm install
```


### Run migration to add tables
```bash
npm run migration:run
```

How to create, run, and delete migration to make db changes in production

```bash
npm run migration:generate --name=YourMigrationName
npm run migration:run
npm run migration:drop
```
## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
